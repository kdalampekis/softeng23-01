Class Diagram

1. Class: NameBasic
Σκοπός: Αντιπροσωπεύει έναν ηθοποιό ή συντελεστή, με βασικές πληροφορίες και το προφίλ του.
Attributes:
nconst: Μοναδικός αναγνωριστικός αριθμός.
primaryName: Όνομα.
birthYear: Έτος γέννησης.
deathYear: Έτος θανάτου, εφόσον ισχύει.
primaryProfession: Κύρια επαγγέλματα.
knownForTitles: Τίτλοι για τους οποίους είναι γνωστός.
img_url_asset: URL εικόνας.
Μέθοδοι:
getFullName(): Επιστρέφει το πλήρες όνομα.
getAge(atYear): Υπολογίζει την ηλικία.
isActive(): Ελέγχει αν είναι ακόμη ενεργός.
getKnownTitles(): Επιστρέφει λίστα γνωστών τίτλων.
getActorProfile(): Υπολογίζει το προφίλ με βάση τα είδη ταινιών.
getMostRecentTitle(): Επιστρέφει την πιο πρόσφατη ταινία.

2. Class: TitleAka
Σκοπός: Αντιπροσωπεύει διαφορετικές εκδόσεις ή εναλλακτικές ονομασίες των τίτλων.
Attributes:
titleId: Αναγνωριστικό τίτλου.
ordering: Σειρά εμφάνισης.
title: Ο τίτλος.
region: Περιοχή.
language: Γλώσσα.
types: Τύπος τίτλου.
attributes: Πρόσθετα χαρακτηριστικά.
isOriginalTitle: Εάν είναι ο πρωτότυπος τίτλος.
Μέθοδοι:
getOriginalTitle(): Επιστρέφει τον πρωτότυπο τίτλο.
getLocalTitle(region, language): Επιστρέφει τον τοπικό τίτλο.

3. Class: TitleBasic
Σκοπός: Αντιπροσωπεύει τις βασικές πληροφορίες ενός τίτλου, όπως ταινίας ή σειράς.
Attributes:
tconst: Μοναδικός αναγνωριστικός αριθμός.
titleType: Τύπος τίτλου.
primaryTitle: Κύριος τίτλος.
originalTitle: Πρωτότυπος τίτλος.
isAdult: Αν είναι για ενήλικες.
startYear: Έτος έναρξης.
endYear: Έτος λήξης για σειρές.
runtimeMinutes: Διάρκεια σε λεπτά.
genres: Είδη.
img_url_asset: URL εικόνας.
Μέθοδοι:
getFormattedTitle(): Επιστρέφει μορφοποιημένο τίτλο.
isCurrentlyAvailable(currentYear): Ελέγχει τη διαθεσιμότητα.
getGenresList(): Επιστρέφει λίστα ειδών.
getRuntimeInHours(): Μετατρέπει τη διάρκεια σε ώρες.
searchByActor(actorId): Αναζητά ταινίες με βάση ηθοποιό.
searchByGenre(genre): Αναζητά ταινίες με βάση είδος.
searchByYear(year): Αναζητά ταινίες με βάση έτος.
getFullDetails(): Επιστρέφει πλήρη στοιχεία της ταινίας.

4. Class: TitleCrew
Σκοπός: Αντιπροσωπεύει το προσωπικό μιας ταινίας ή σειράς, όπως σκηνοθέτες και σεναριογράφοι.
Attributes:
tconst: Μοναδικός αναγνωριστικός αριθμός τίτλου.
directors: Σκηνοθέτες.
writers: Σεναριογράφοι.
Μέθοδοι:
getDirectors(): Επιστρέφει λίστα σκηνοθετών.
getWriters(): Επιστρέφει λίστα σεναριογράφων.
addCrewMember(nconst, role): Προσθέτει μέλος προσωπικού.
getMostHighlyRatedTitle(): Επιστρέφει την υψηλότερα βαθμολογημένη ταινία του συντελεστή.
searchByCrewMember(crewId): Αναζητά ταινίες με βάση τον συντελεστή.

5. Class: TitleEpisode
- Σκοπός: Αντιπροσωπεύει μεμονωμένα επεισόδια τηλεοπτικών σειρών, παρέχοντας λεπτομέρειες όπως αριθμός σεζόν και επεισοδίου.
- Attributes:
  - `tconst`: Μοναδικός αναγνωριστικός αριθμός του επεισοδίου.
  - `parentTconst`: Αναγνωριστικός αριθμός της γονικής σειράς.
  - `seasonNumber`: Αριθμός της σεζόν.
  - `episodeNumber`: Αριθμός του επεισοδίου.
- Μέθοδοι:
  - `getParentSeries()`: Επιστρέφει τη γονική σειρά ως `TitleBasic` αντικείμενο.
  - `getSeasonAndEpisode()`: Επιστρέφει ένα tuple με τον αριθμό της σεζόν και του επεισοδίου.

6. Class: TitlePrincipal
- Σκοπός: Αντιπροσωπεύει τους βασικούς συντελεστές μιας ταινίας ή επεισοδίου, περιλαμβάνοντας ηθοποιούς, σκηνοθέτες, παραγωγούς και άλλες βασικές θέσεις.
- Attributes:
  - `tconst`: Αναγνωριστικός αριθμός του τίτλου.
  - `ordering`: Σειρά εμφάνισης.
  - `nconst`: Αναγνωριστικός αριθμός του ατόμου.
  - `category`: Κατηγορία (π.χ. ηθοποιός, σκηνοθέτης).
  - `job`: Ειδική θέση εργασίας.
  - `characters`: Χαρακτήρες που υποδύθηκε.
  - `img_url_asset`: URL εικόνας.
- Μέθοδοι:
  - `getPersonDetails()`: Επιστρέφει το `NameBasic` αντικείμενο του ατόμου.
  - `getRole()`: Επιστρέφει τον ρόλο του ατόμου στην ταινία.
  - `getCharacters()`: Επιστρέφει λίστα με τους χαρακτήρες.
  - `getBi

ography()`: Επιστρέφει το βιογραφικό του ηθοποιού/συντελεστή.
  - `getTopRatedTitles(N)`: Επιστρέφει λίστα με τις N πιο υψηλά βαθμολογημένες ταινίες του ηθοποιού/συντελεστή.

7. Class: TitleRating
- Σκοπός: Παρέχει πληροφορίες για τις βαθμολογίες των ταινιών, όπως ο μέσος όρος και ο αριθμός των ψήφων.
- Attributes:
  - `tconst`: Αναγνωριστικός αριθμός του τίτλου.
  - `averageRating`: Μέσος όρος βαθμολογίας.
  - `numVotes`: Αριθμός ψήφων.
- Μέθοδοι:
  - `getRatingDetails()`: Επιστρέφει λεπτομέρειες βαθμολογίας.
  - `isHighlyRated(threshold)`: Ελέγχει αν η βαθμολογία υπερβαίνει ένα καθορισμένο όριο.

ER:

Entity: NameBasic

Αντιπροσωπεύει έναν ηθοποιό ή συντελεστή.
Σχέση με το TitlePrincipal: Ένας NameBasic μπορεί να εμφανίζεται σε πολλά TitlePrincipal.
Entity: TitleBasic

Αντιπροσωπεύει τη βασική πληροφορία για κάθε τίτλο ταινίας ή επεισοδίου.
Σχέση με το TitleRating: Ένας TitleBasic έχει ένα TitleRating.
Σχέση με το TitlePrincipal: Ένας TitleBasic μπορεί να έχει πολλούς TitlePrincipal.
Entity: TitleAka

Αντιπροσωπεύει εναλλακτικούς τίτλους για ταινίες.
Σχέση με το TitleBasic: Ένας TitleAka αντιστοιχίζεται σε ένα TitleBasic.
Entity: TitleCrew

Αντιπροσωπεύει τα μέλη της ομάδας παραγωγής (σκηνοθέτες, σεναριογράφοι).
Σχέση με το TitleBasic: Ένας TitleCrew αντιστοιχίζεται σε ένα TitleBasic.
Entity: TitleEpisode

Αντιπροσωπεύει επεισόδια τηλεοπτικών σειρών.
Σχέση με το TitleBasic: Ένας TitleEpisode αντιστοιχίζεται σε ένα TitleBasic (γονική σειρά).
Entity: TitlePrincipal

Αντιπροσωπεύει τους κύριους συντελεστές μιας ταινίας.
Σχέση με το TitleBasic και NameBasic.
Entity: TitleRating

Αντιπροσωπεύει τις βαθμολογίες των ταινιών.
Σχέση με το TitleBasic: Ένας TitleRating αντιστοιχίζεται σε ένα TitleBasic.
Entity: User

Αντιπροσωπεύει τους χρήστες της εφαρμογής.
Δεν έχει άμεση σχέση με τις άλλες κλάσεις αλλά μπορεί να συμμετέχει σε διάφορες ενέργειες (π.χ. αξιολόγηση, σχολιασμός)

Component Diagram

User Interface (UI) Component

Περιλαμβάνει όλα τα διεπαφικά στοιχεία για την αλληλεπίδραση με τον χρήστη.
Search Engine Component

Διαχειρίζεται την αναζήτηση ταινιών, συντελεστών και άλλων σχετικών πληροφοριών.
Database Management Component

Αποθηκεύει και διαχειρίζεται δεδομένα ταινιών, συντελεστών, χρηστών, και αξιολογήσεων.
Rating and Review System Component

Διαχειρίζεται τις βαθμολογίες και τις κριτικές των ταινιών.
Authentication Component

Υπεύθυνο για την ασφάλεια και την πιστοποίηση των χρηστών.
Data Analysis Component

Αναλύει τα δεδομένα για την παραγωγή στατιστικών και εισηγήσεων.
External API Integration Component

Ενσωματώνει εξωτερικές πηγές δεδομένων και API για επιπλέον πληροφορίες.
Notification Component

Διαχειρίζεται ειδοποιήσεις και ενημερώσεις προς τους χρήστες.
User Preferences Component

Διαχειρίζεται τις προτιμήσεις και τις ρυθμίσεις των χρηστών.
Σχέσεις μεταξύ των Components:
Το UI Component αλληλεπιδρά με όλα τα άλλα συστατικά, παρέχοντας τη διεπαφή για τις ενέργειες του χρήστη.
Το Search Engine συνδέεται άμεσα με το Database Management για την ανάκτηση και ενημέρωση δεδομένων.
Το Rating and Review System και το Data Analysis συνδέονται με το Database Management για την ανάλυση και παρουσίαση δεδομένων.
Το Authentication Component εξασφαλίζει την ασφάλεια και την ταυτοποίηση των διαδικασιών.
Το External API Integration επεκτείνει τις δυνατότητες του συστήματος με εξωτερικές πηγές.
Το Notification και το User Preferences ενισχύουν την εμπειρία χρήστη.

Deployment diagram

Υπάρχει ένας εξυπηρετητής (Server) που φιλοξενεί την εφαρμογή "ntuaflix" και τη βάση δεδομένων (Database).

Οι χρήστες έχουν πρόσβαση στην εφαρμογή μέσω του περιηγητή τους (Browser).

Η εφαρμογή "ntuaflix" αποθηκεύει και διαχειρίζεται τα δεδομένα των ταινιών, των χρηστών και των αξιολογήσεων στη βάση δεδομένων.

Υπάρχει ένα δίκτυο (Network) που συνδέει τον εξυπηρετητή με τους χρήστες.

Η εφαρμογή "ntuaflix" επικοινωνεί με εξωτερικές υπηρεσίες (External Services) για την λήψη επιπλέον πληροφοριών σχετικά με τις ταινίες.

Η εφαρμογή "ntuaflix" χρησιμοποιεί έναν εξωτερικό εξυπηρετητή SMTP (SMTP Server) για την αποστολή ειδοποιήσεων και επιβεβαιώσεων στους χρήστες.
